# Emotion-Detector
Developed a Live Emotion Detector using Python and TensorFlow that captures real-time facial expressions through a webcam and classifies emotions such as happy, sad, angry, surprised, and neutral. The system leverages a convolutional neural network (CNN) for accurate emotion recognition and provides instant feedback on detected emotions.
# Live Emotion Detector

A real-time emotion detection system built using Python and TensorFlow that identifies human emotions through facial expressions captured via webcam.

## Features
- Real-time face detection using webcam
- Emotion classification using CNN model
- Detects emotions such as happy, sad, angry, surprised, and neutral

## Technologies Used
- Python
- TensorFlow
- OpenCV
- NumPy

## Project Files
- train_model.py – used to train the emotion recognition model
- emotion_detector.py – runs the live emotion detection
- emotion_model.h5 – trained model file
